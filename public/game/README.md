# Volcano Escape Art Assets

These project-owned raster assets were generated for the hackathon presentation on 2026-07-18 using the built-in OpenAI image generation tool. They contain no third-party logos, text, or imported game IP.

- `volcano-escape-bg.png`: 16:9 hand-painted volcanic valley, lava bridge, and cyan escape portal. The scene flips it horizontally so mission progress travels toward the portal.
- `runner.png`: side-view fitness hero generated on a flat chroma-key background, then converted locally to an alpha PNG with the Codex image-generation skill helper.
- `volcanic-guardian.png`: original Ash Titan boss generated on a flat chroma-key background on 2026-07-20, then converted locally to an alpha PNG. It contains no logo, text, trademark, or imported game IP.
- `scout-guide-sheet-source.png`: original 2×2 Scout character sheet generated on 2026-07-21 with a flat chroma background.
- `scout-guide-sheet.png`: transparent production sheet derived from that source with the image-generation skill chroma-key helper. The quadrants show Scout waving, checking a compass, pointing ahead, and cheering.
- `src/app/icon.png`: opaque square application icon generated on 2026-07-21. Its mountain silhouette, cyan summit route, gold escape portal, and ember base remain recognizable without text at small sizes.

The prompts requested premium 2D indie-game art in the existing deep-navy, violet, ember-orange, cyan, and gold palette. UI, characters, logos, and text were excluded from the background; the runner prompt requested an isolated blue/cyan athletic hero with an orange cape and no text or extra objects. Scout was prompted as the same friendly, gender-neutral young-adult trail guide in every pose, wearing a deep-navy jacket, cyan vest accents, orange scarf, dark trail pants, trail shoes, and a compact backpack, with clean silhouette separation and no text or logo.

These assets are presentation-only. Replacing them must not change mission snapshots, pose events, scoring, progress, or results.
