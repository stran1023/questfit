# AI Fitness Escape --- Product Requirements Document (PRD)

## 1. Product Vision

AI Fitness Escape is an AI-powered fitness adventure where personalized
workouts are transformed into interactive game levels. Players exercise
in front of a webcam, and real-world movements drive in-game
progression.

## 2. Problem

Traditional home workouts are repetitive and lack motivation. Existing
fitness apps track activity but rarely make exercising entertaining.

## 3. Target Users

-   Students
-   Busy professionals
-   Casual fitness beginners
-   Home workout enthusiasts

## 4. Goals

-   Increase workout engagement
-   Personalize sessions using AI
-   Make exercise feel like gameplay
-   Encourage long-term habits through progression

## 5. Core User Flow

1.  Sign in or continue as Guest
2.  Complete calibration
3.  AI generates workout
4.  Workout becomes a game level
5.  Play using body movements
6.  Receive AI coaching
7.  Save score, XP, streak and leaderboard

## 6. MVP Features

-   Google login + Guest mode
-   AI workout generation
-   Level Compiler
-   Pose detection (MediaPipe)
-   Phaser adventure gameplay
-   Session summary
-   Weekly leaderboard
-   XP and daily streak

## 7. Stretch Goals

-   Multiple worlds
-   Boss battles
-   Achievements
-   Friends leaderboard
-   Adaptive difficulty
-   AI posture feedback

## 8. Success Metrics

-   Workout completion rate
-   Average session duration
-   Daily retention
-   Streak continuation
-   Leaderboard participation

## 9. Non-functional Requirements

-   Browser-based
-   \<100ms pose response target
-   Mobile-friendly UI
-   Secure authentication
-   Privacy-first (no webcam upload)

## 10. MVP Scope (Hackathon)

Focus on one polished gameplay loop with AI-generated workout, one
adventure theme, leaderboard, and AI coaching.
